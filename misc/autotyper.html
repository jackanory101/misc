AutoType — User Guide

AutoType is a typing simulator. It takes text from a file and types it into another program (like Vim, Notepad, or a terminal) while playing key click sounds.

You can run it as:

  • A Python script → autotype.py
  • A standalone executable → autotype.exe

The tool makes typing look and sound natural by:

  • Adding slight randomness to speed.
  • Using different sounds for normal keys vs. space/Enter.
  • Making occasional typos (then correcting them).
  • Adding pauses, like a real person thinking.

How to Use

  1. Put your text into a file (default: text_to_type.txt).

  2. Run AutoType:

    python autotype.py

    or if you have the executable:

    autotype.exe

    You can also specify a file:

    autotype.py mytext.txt

  3. After starting, you’ll have a few seconds (FOCUS_DELAY) to switch to the target window.

  4. AutoType will type everything automatically.

Configuration (config.json)

AutoType is fully customizable. You can change typing speed, randomness, pauses, and sounds.

Setting What it does Example
BASE_INTERVAL Base delay between keystrokes (seconds). 0.03
INTERVAL_VARIATION Random variation multiplier (min, max). [0.9, 1.3]
NEWLINE_PAUSE Pause after pressing Enter (min, max seconds). [0.4, 0.8]
TYPO_PROBABILITY Chance of making and correcting a typo. 0.02
THINKING_PROBABILITY Chance of a longer “thinking pause”. 0.01
THINKING_PAUSE_RANGE Duration of thinking pauses (min, max seconds). [1.0, 2.5]
TYPO_CHARS Letters used when making a typo. "abcdefghijklmnopqrstuvwxyz"
FOCUS_DELAY Seconds to wait before typing starts. 2
DEFAULT_FILE Text file used if none is specified. "text_to_type.txt"
KEY_SOUND_FILE Sound for normal keys. "sounds/key_click.wav"
SPACE_ENTER_SOUND_FILE Sound for Space and Enter. "sounds/spacebar.wav"

Notes

  • Sounds are optional — if missing, typing is silent.

  • Pauses:

    • NEWLINE_PAUSE happens after Enter.
    • THINKING_PAUSE_RANGE only happens between words or punctuation (never in the middle of a word).

  • Typing speed is variable but always realistic — it won’t go too fast or too slow.
